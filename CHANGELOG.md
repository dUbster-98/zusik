# Changelog

형식은 [Keep a Changelog](https://keepachangelog.com/ko/), 버전은 [SemVer](https://semver.org)를 따른다.

## [1.0.0] - 2026-06-30

첫 공개 릴리스. 한국·미국 주식과 암호화폐를 시장 시간에 맞춰 24시간 자동매매하는 봇이다.
LLM 멀티 애널리스트 분석과 Monte Carlo 시뮬레이션으로 판단하고, 실거래 결과로 파라미터를
스스로 보정한다. 실거래에 쓰는 코드이므로 교육·연구 목적으로 제공하며 수익을 보장하지 않는다.

### 매매와 종목 선택
- 멀티 마켓 운용. 한국(KIS·토스), 미국 주식, 업비트 암호화폐를 각 시장 시간에 맞춰 돌린다.
  미국 매매는 `us_enabled` 로 끌 수 있다.
- 증권사 선택. `.env` 의 `BROKER` 로 한국투자증권과 토스증권 중 고른다. 둘 다 국내·미국을
  지원하고 라이브로 검증했다.
- 선택형 스크리닝. Monte Carlo, 모멘텀, 추세, 저변동 중에서 방식을 고른다. 상대강도 게이트와
  레짐 틸트(하락장 방어주, 상승장 모멘텀), 로테이션, 이벤트 섹터 부스트로 국면에 맞춘다.
- 빠른 진입. 5분 사이클 사이의 급등 돌파와 과매도 반등을 추가 LLM 호출 없이 잡는다. 실시간
  WebSocket 틱으로 익절과 진입을 보조한다.

### AI 분석
- 멀티 애널리스트. 펀더멘털·센티멘트·퀀트·종합 네 관점이 같은 출력 계약(JSON)으로 답하고
  성과 가중 투표로 합친다. provider(claude·codex·agy)가 달라도 결과 형식이 같다.
- 가중치 하네스. 확신도와 합의, 과거 적중률이 매수 금액으로 이어진다. 자세한 구조는
  [docs/LLM.md](docs/LLM.md)에 정리했다.
- 비용 관리. 용도별로 모델을 나누고 provider를 라운드로빈으로 돌리며, 요금제별 하루 한도를
  둔다. 모든 CLI가 막히면 로컬 퀀트 전략으로 매매를 잇는다. 로컬 LLM(Ollama)도 쓸 수 있다.

### 안전망
- 다층 방어. 추세 필터, 변동성 비례 급락 감지, 트레일링, 본전 보호, churn 가드가 LLM 없이
  즉시 작동한다.
- 급변동 가드. 개장 직후 시초 추격과 지수 급락에서 신규 진입을 멈추고 헷지를 연다. 보유 종목은
  자르지 않는다.
- 자본 보호. 종목별 하드스톱, 일일 손실 한도, 드로우다운 기반 사이징 축소.
- 주문 관문. 모든 주문이 `OrderSafetyValidator` 를 통과해 초과매도·워시트레이딩·스푸핑·계좌
  드레인을 fail-closed 로 막는다.
- 위험 종목 감지. 거래정지·상장폐지를 KRX 지정 용어로 정밀하게 잡고, 돌이킬 수 없는 자동매도는
  종목이 거래 중이면 보류하고 사람 확인을 요청한다. 종목명은 KIS 상품마스터에서 권위 있게 가져온다.

### 인버스 헷지
- 상황 적응. 지속적 위기에서만 발동해 단발 급락 휩쏘를 피한다. 기초지수가 실제로 빠질 때만 사서
  시장별 무차별 매수를 막는다.
- 수익화. 순익이 임계를 넘으면 바로 익절하고, 마감 임박이나 반등 시 락인한다. 익절 임계는 고정이
  아니라 실거래 성과로 스스로 조정한다.

### 자가학습
- 청산 파라미터. 다년 일봉 walk-forward 백테스트로 수익 사다리와 본전 giveback 을 고른다.
- 손실 보유 바닥. 손실 컷이 조기였는지 정당했는지를 사후데이터로 측정해 보유 바닥을 조정한다.
  하드스톱은 건드리지 않는다.
- 사후 분석. 매도 타이밍과 종목 선택의 초과수익을 데이터로 평가해 리포트에 싣는다.

### 운영과 알림
- 24/7 systemd 서비스와 워치독. 죽음, 멈춤(hang), LLM 다운을 감지해 즉시 알린다.
- 통합 상태 스냅샷(`status.json`)과 `--status` 대시보드, 별도 웹 뷰.
- 멀티 메신저. Discord·Telegram·Slack 동시 알림과 원격 명령. 봇이 죽었을 때를 대비해 웹훅
  알림을 권장한다.
- 리포트. 월간 성과와 투자결과 종합(HTML·PDF), 종목별 손익·승률·매도 패턴.
- 무결성 트립와이어와 pre-commit 자동 갱신으로 소스 변조를 감지한다.

### 설치
- 대화형 `setup.sh` 마법사. 환경 세팅, API 키 입력, 증권사·미국매매·AI 요금제 선택, 위험 고지를
  차례로 안내한다.
- Python 3.8+. 핵심 의존성만 쓰고 해시로 핀한다(`requirements.lock`).
