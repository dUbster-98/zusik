## [1.0.2] - 2026-07-02

매수가 조용히 멈추던 경계들을 걷어내고, 우회 매수 경로를 일반 게이트·회계와 정합시키고,
외부 보안 리뷰(총 7라운드)를 반영한 안정화 릴리스.

### 고침
- **신중(cautious) 장세에서 로컬 전략 매수가 전면 정지되던 문제.** 로컬 퀀트(adaptive)의
  확신도가 RSI 스텁이라 중립 구간에서 0.5에 고정돼, 장전 요구치(0.55)를 구조적으로 못 넘었다.
  전략 선택에 이미 쓰던 백테스트 종합 점수(수익·샤프·승률)를 확신도로 변환해 검증된 edge가
  있는 전략은 게이트를 통과한다. 승률이 아닌 점수를 쓰는 이유: 모멘텀 전략은 승률 40%대로
  비대칭 수익을 내는 게 정상이라 승률 매핑으로는 강한 edge도 걸러졌다.
- **소액 계좌가 매매를 포기하던 경계들.** 최소주문×2 경계 계좌가 올인 판정에서 탈락하던 문제,
  동적 스로틀이 투자금을 최소주문 밑으로 누르던 문제에 이어, 주가가 (현금×95%, 현금] 구간이면
  0주로 조용히 포기하던 마지막 갭을 1주 상향으로 막았다(KR/US 공통).
- **우회 매수 경로의 게이트·회계 정합화.** US "보유 0 강제매수"와 KR/US 잔여현금 소진 매수가
  장전 리스크오프·재진입 차단·보수 모드를 건너뛰고, 실제 매수 종목이 아닌 원래 후보 기준으로
  게이트를 보던 문제를 수정. 성공한 매수는 포지션 상태(본전 보호·트레일링·피라미딩)에 빠짐없이
  기록된다.
- **수동 명령(/매수·/매도)이 봇 회계 밖에 있던 문제.** 체결 시 tracker/포지션에 "수동 명령"으로
  기록해 실현손익이 스냅샷 추정치로 어긋나지 않는다. 수동 진입의 손익은 자동 전략 학습과
  분리된 버킷으로 집계해 전략 가중치를 오염시키지 않는다.
- **인버스 진입 기준을 함의 지수 낙폭으로.** ETF 자체 등락이 아니라 레버리지를 정규화한 기초지수
  낙폭이 급락 수준일 때만 진입 — 지수는 멀쩡한데 인버스 차트만 흔들릴 때의 무차별 매수를 차단.
- **US 유휴 진입 죽은 코드 삭제.** 전 종목 매도신호일 때 "최약 SELL 소량 진입" 분기가 빈 리스트를
  정렬하고 있어 실행된 적이 없었다 — 신호 존중 설계에 맞게 제거하고 현금 대기로 단순화.
- 원격 업데이트 명령이 재시작 실패(sudo 거부 등)를 성공으로 오보하던 문제 — 실패를 정확히
  보고한다("구버전 계속 실행 중" 경고).

### 추가
- **진입 유형별 ROI 관측.** 일반/잔금소진/강제매수/수동 진입의 건수·승률·손익을
  `scripts/entry_roi.py` 로 즉시 확인하고, 월간 리포트(텍스트·Discord embed·HTML)에도 결산 월과
  같은 기간축으로 "진입 유형별 손익"이 상시 포함된다. 잔여현금 소진 같은 보조 진입이 국면 변화로
  음수 전환하면 리포트에서 바로 드러난다.
- LLM sonnet 티어를 claude-sonnet-5 로 상향(CLI 매핑 + API 폴백 기본값).

### 보안
- **Discord 명령 권한 fail-closed.** `DISCORD_OWNER_ID` 미설정 시 서버 관리자 폴백 대신 거부하고,
  빈 값이어도 시작 크래시 없이 안전 파싱한다. 당일 매수 게이트를 바꾸는 `/장전분석` 도 소유자
  전용으로 전환.
- **모의투자 기본값.** `.env.example` 이 `KIS_VIRTUAL=true` 로 시작해, 예제를 복사해 키만 채워도
  실계좌 매매가 시작되지 않는다. 실거래는 명시적 opt-in.
- 구식 관리자 폴백과 원격 업데이트 실행이 남아 있던 레거시 스크립트(`scripts/notify_update.py`)
  삭제.
- 릴리스 도구 안전화: 태그 충돌 선검사(로컬 릴리스 브랜치 오염 방지), 릴리스 생성 실패 은폐 제거
  ("이미 존재"만 허용), 실패 경로를 포함해 항상 개발 계정 복원. 보안 신고 링크를 공개 저장소
  브랜치(main)로 수정.
- 업데이트 감지가 `origin/master` 하드코딩이라 main 이 기본 브랜치인 클론에서 원격 커밋을 못
  보던 문제 — 추적 브랜치를 동적으로 감지한다.

## [1.0.1] - 2026-06-30

### 추가
- 한국 매매 on/off 토글(`kr_enabled`). 미국만/암호화폐만 운용하려는 사용자를 위해 한국 주식을
  끌 수 있다. `setup.sh` 마법사가 묻고, `configtool.py set kr_enabled false` 로도 끈다.
  (기존 `us_enabled` 와 대칭 — 둘 다 끄면 암호화폐만)
- 월간 리포트에 **매도 패턴별 손익** 섹션 추가. 종합 리포트와 같은 축으로 "무엇이 돈을 벌었나"를
  월 단위로 보여준다(패턴 한국어 라벨·건수·승률·손익).

### 문서
- README 첫 블록을 더 직관적으로 개편 — 한 줄 가치 제안 + 스캔 가능한 볼드 불릿(구체적 수치).
- 빠른 시작 "준비물"에 토스증권도 쓸 수 있음을 명시(`BROKER=toss`, 국내·미국 지원).
- 시장별 끄기 안내 추가.

## [1.0.0] - 2026-06-30

첫 공개 릴리스. 한국·미국 주식과 암호화폐를 시장 시간에 맞춰 24시간 자동매매하는 봇이다.
LLM 멀티 애널리스트 분석과 Monte Carlo 시뮬레이션으로 판단하고, 실거래 결과로 파라미터를
스스로 보정한다. 실거래에 쓰는 코드이므로 교육·연구 목적으로 제공하며 수익을 보장하지 않는다.

### 매매와 종목 선택
- 멀티 마켓 운용. 한국(KIS·토스), 미국 주식, 업비트 암호화폐를 각 시장 시간에 맞춰 돌린다.
  미국 매매는 `us_enabled` 로 끌 수 있다.
- 증권사 선택. `.env` 의 `BROKER` 로 한국투자증권과 토스증권 중 고른다. 둘 다 국내·미국을
  지원하고 라이브로 검증했다.
- 선택형 스크리닝. Monte Carlo, 모멘텀, 추세, 저변동 중에서 방식을 고른다. 상대강도 게이트와
  레짐 틸트(하락장 방어주, 상승장 모멘텀), 로테이션, 이벤트 섹터 부스트로 국면에 맞춘다.
- 빠른 진입. 5분 사이클 사이의 급등 돌파와 과매도 반등을 추가 LLM 호출 없이 잡는다. 실시간
  WebSocket 틱으로 익절과 진입을 보조한다.

### AI 분석
- 멀티 애널리스트. 펀더멘털·센티멘트·퀀트·종합 네 관점이 같은 출력 계약(JSON)으로 답하고
  성과 가중 투표로 합친다. provider(claude·codex·agy)가 달라도 결과 형식이 같다.
- 가중치 하네스. 확신도와 합의, 과거 적중률이 매수 금액으로 이어진다. 자세한 구조는
  [docs/LLM.md](docs/LLM.md)에 정리했다.
- 비용 관리. 용도별로 모델을 나누고 provider를 라운드로빈으로 돌리며, 요금제별 하루 한도를
  둔다. 모든 CLI가 막히면 로컬 퀀트 전략으로 매매를 잇는다. 로컬 LLM(Ollama)도 쓸 수 있다.

### 안전망
- 다층 방어. 추세 필터, 변동성 비례 급락 감지, 트레일링, 본전 보호, churn 가드가 LLM 없이
  즉시 작동한다.
- 급변동 가드. 개장 직후 시초 추격과 지수 급락에서 신규 진입을 멈추고 헷지를 연다. 보유 종목은
  자르지 않는다.
- 자본 보호. 종목별 하드스톱, 일일 손실 한도, 드로우다운 기반 사이징 축소.
- 주문 관문. 모든 주문이 `OrderSafetyValidator` 를 통과해 초과매도·워시트레이딩·스푸핑·계좌
  드레인을 fail-closed 로 막는다.
- 위험 종목 감지. 거래정지·상장폐지를 KRX 지정 용어로 정밀하게 잡고, 돌이킬 수 없는 자동매도는
  종목이 거래 중이면 보류하고 사람 확인을 요청한다. 종목명은 KIS 상품마스터에서 권위 있게 가져온다.

### 인버스 헷지
- 상황 적응. 지속적 위기에서만 발동해 단발 급락 휩쏘를 피한다. 기초지수가 실제로 빠질 때만 사서
  시장별 무차별 매수를 막는다.
- 수익화. 순익이 임계를 넘으면 바로 익절하고, 마감 임박이나 반등 시 락인한다. 익절 임계는 고정이
  아니라 실거래 성과로 스스로 조정한다.

### 자가학습
- 청산 파라미터. 다년 일봉 walk-forward 백테스트로 수익 사다리와 본전 giveback 을 고른다.
- 손실 보유 바닥. 손실 컷이 조기였는지 정당했는지를 사후데이터로 측정해 보유 바닥을 조정한다.
  하드스톱은 건드리지 않는다.
- 사후 분석. 매도 타이밍과 종목 선택의 초과수익을 데이터로 평가해 리포트에 싣는다.

### 운영과 알림
- 24/7 systemd 서비스와 워치독. 죽음, 멈춤(hang), LLM 다운을 감지해 즉시 알린다.
- 통합 상태 스냅샷(`status.json`)과 `--status` 대시보드, 별도 웹 뷰.
- 멀티 메신저. Discord·Telegram·Slack 동시 알림과 원격 명령. 봇이 죽었을 때를 대비해 웹훅
  알림을 권장한다.
- 리포트. 월간 성과와 투자결과 종합(HTML·PDF), 종목별 손익·승률·매도 패턴.
- 무결성 트립와이어와 pre-commit 자동 갱신으로 소스 변조를 감지한다.

### 설치
- 대화형 `setup.sh` 마법사. 환경 세팅, API 키 입력, 증권사·미국매매·AI 요금제 선택, 위험 고지를
  차례로 안내한다.
- Python 3.8+. 핵심 의존성만 쓰고 해시로 핀한다(`requirements.lock`).
